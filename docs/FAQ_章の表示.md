# FAQ: 章の追加後に反映されない問題

## 問題: 章を追加したのに公開サイトに表示されない

### 確認事項

1. **章は実際に表示されています**
   - https://kampo.belong2jazz.workers.dev/texts/shanghan-lun を確認
   - 「太陽」という章は2番目に表示されています

2. **表示順序について**
   - 章は `chapter_order`（章番号）の順に表示されます
   - 同じ章番号が複数ある場合、順序が期待通りにならない可能性があります

### 解決方法

#### 方法1: ブラウザキャッシュをクリア

1. ブラウザで **Ctrl+Shift+R**（Windows）または **Cmd+Shift+R**（Mac）を押して強制リロード
2. またはブラウザの設定からキャッシュをクリア

#### 方法2: 章番号を調整

現在、以下の章が`chapter_order=1`で重複しています:
- `chapter-1`: 辨太陽病脈證併治上（元からあった章）
- `shanghan-lun-1`: 太陽（新しく追加した章）

**修正手順**:

1. 管理画面にログイン: https://kampo.belong2jazz.workers.dev/admin/login
2. 「章の管理」をクリック
3. 傷寒論を選択
4. 「太陽」章の「編集」をクリック
5. 章番号を変更（例: `4` にする）
6. 「更新する」をクリック

これで、章は以下の順序で表示されます:
1. 辨太陽病脈證併治上
2. 辨太陽病脈證併治中
3. 辨太陽病脈證併治下
4. 太陽

#### 方法3: 元の章を削除または章番号を変更

元の3つの章（chapter-1, chapter-2, chapter-3）を削除または章番号を変更して、
新しい章を1番目に持ってくることもできます。

**注意**: 削除は取り消せないので、慎重に行ってください。

### 現在の状況（2025-11-11時点）

傷寒論の章一覧:
- **順序1**: 辨太陽病脈證併治上 (chapter-1)
- **順序1**: 太陽 (shanghan-lun-1) ← **新しく追加された章**
- **順序2**: 辨太陽病脈證併治中 (chapter-2)
- **順序3**: 辨太陽病脈證併治下 (chapter-3)

### 推奨される章番号の設定

傷寒論を最初から構築する場合:
1. 太陽 - chapter_order: 1
2. （その他の章を順次追加）

または、既存の章を残す場合:
1. 辨太陽病脈證併治上 - chapter_order: 1
2. 辨太陽病脈證併治中 - chapter_order: 2
3. 辨太陽病脈證併治下 - chapter_order: 3
4. 太陽 - chapter_order: 4 ← **修正推奨**

### 確認方法

1. https://kampo.belong2jazz.workers.dev/texts/shanghan-lun にアクセス
2. 「全章数」が **4** と表示されていることを確認
3. 章一覧に「太陽」が表示されていることを確認
